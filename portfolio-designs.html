About the Projects & Tools

I implement my project designs into real code using VS Code as the primary editor. For front-end projects, I use HTML and TailwindCSS to create modern and responsive interfaces.

For full-stack projects, I integrate various technologies. The backend is built with Express.js, while PostgreSQL serves as the database. Dynamic rendering is powered by Handlebars (HBS). Secure login is implemented with bcryptjs and express-session, and image upload is handled by Multer. The project is hosted on Railway.

This approach demonstrates my ability to build complete web applications from front-end to back-end, and my understanding of different tools and frameworks.

Desain Portfolio & Proyek

Berikut adalah desain mockup dan detail proyek-proyek yang saya bangun untuk portofolio. Saya menggunakan tools dan teknologi modern untuk menghasilkan tampilan yang bersih, responsif, dan fungsional.

Aplikasi Manajemen Data

Proyek ini merupakan aplikasi web lengkap yang dibangun dengan tumpukan teknologi modern. Aplikasi ini menampilkan manajemen data dengan otentikasi pengguna, unggah gambar, dan tampilan dinamis.

Teknologi yang Digunakan:

  • Express.js (Node.js) - Backend Framework
  • PostgreSQL - Database relasional
  • Handlebars (HBS) - Template Engine
  • Railway - Cloud Hosting
  • bcryptjs + express-session - Sistem Login
  • Multer - Mengunggah gambar
  • TailwindCSS - Styling Modern

Website Portofolio Responsif

Proyek ini adalah halaman web statis yang berfokus pada desain antarmuka pengguna (UI) dan pengalaman pengguna (UX) yang menarik. Dibuat murni menggunakan **HTML** dan **Tailwind CSS**, memastikan desain yang bersih, responsif, dan mudah dikelola.

Tentang Proyek & Tools

Saya mengimplementasikan desain proyek-proyek saya ke dalam kode menggunakan VS Code sebagai code editor. Untuk proyek-proyek front-end, saya menggunakan HTML dan Tailwind CSS untuk menciptakan tampilan yang modern dan responsif.

Sementara itu, untuk proyek full-stack, saya menggabungkan berbagai teknologi. Backend-nya dibangun dengan Express.js, dengan PostgreSQL sebagai database. Tampilan dinamis dihasilkan oleh Handlebars (HBS). Saya juga menerapkan sistem login yang aman dengan bcryptjs dan express-session serta fitur unggah gambar menggunakan Multer. Proyek ini di-hosting di Railway.

Dengan pendekatan ini, saya dapat menunjukkan kemampuan saya dalam membangun aplikasi web yang komprehensif, dari front-end hingga back-end, serta pemahaman saya dalam menggunakan berbagai tool dan framework.
